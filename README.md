# Portfólio | James Marco Souza Pereira

Portfólio de Analista de Dados — React, TypeScript, Vite, Tailwind CSS, shadcn/ui.

## Como rodar

```sh
npm install
npm run dev
```

## Scripts

- `npm run dev` — servidor de desenvolvimento
- `npm run build` — build de produção
- `npm run preview` — preview do build
- `npm run build:gh` — build com base `/Projeto/` (GitHub Pages)

## Tecnologias

- Vite
- TypeScript
- React
- Tailwind CSS
- shadcn-ui
- Framer Motion
